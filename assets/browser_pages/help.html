TBrowser Help

Press / to open the command palette. Below are the available commands:

Command Action
/nt:<url> Open a new tab to the URL
/pt:<url> Open a new private tab (excluded from history)
/t:<url> Open the URL in the current tab
/s:<text> Search <text> in the current tab (Google)
/ts:<text> Search <text> in a new tab (Google)
/nw Open a new window
/hist Open a history page
/help Show this help
/capture Capture a screenshot of the visible page
